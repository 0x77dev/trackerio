# Tracker.io

> Track people via their devices.

## Usage

## Install

With [npm](https://npmjs.org/) installed, run

```
$ npm install -g @0x77/trackerio
```

With [yarn](https://yarnpkg.org/) installed, run

```
$ yarn global add @0x77/trackerio
```
